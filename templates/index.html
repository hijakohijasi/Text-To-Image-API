API Documentation

Endpoints
  • POST /api/generate
    Generate image from text
  • GET /api/health
    Health check
  • GET /api/styles
    Available styles
  • GET /api/sizes
    Available sizes
Rate Limits
  • 5 requests per minute
  • 50 requests per hour
  • 200 requests per day
Response Format

Images are returned as base64-encoded strings in JSON format.

Example Request

cURL GET
curl "{{ request.url_root }}api/generate?prompt=A%20serene%20mountain%20landscape%20at%20sunset&style=realistic&size=medium"
cURL POST
curl -X POST \
  {{ request.url_root }}api/generate \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A serene mountain landscape at sunset",
    "style": "realistic",
    "size": "medium"
  }'
Python GET
import requests

response = requests.get(
    "{{ request.url_root }}api/generate",
    params={
        "prompt": "A serene mountain landscape at sunset",
        "style": "realistic",
        "size": "medium"
    }
)

data = response.json()
if data.get("success"):
    image_base64 = data["image"]
Python POST
import requests

response = requests.post(
    "{{ request.url_root }}api/generate",
    json={
        "prompt": "A serene mountain landscape at sunset",
        "style": "realistic",
        "size": "medium"
    }
)

data = response.json()
if data.get("success"):
    image_base64 = data["image"]
